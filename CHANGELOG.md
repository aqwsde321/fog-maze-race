# Changelog

이 문서는 Fog Maze Race의 주요 변경 이력을 기록합니다.

## [Unreleased]

### Added

- 방 안에서 최근 레이스 결과를 다시 확인할 수 있는 로그 패널 추가

### Changed

- 레이스 로그 버튼을 우측 액션 패널로 이동하고 라벨을 `로그`로 정리
- 가짜 골 `쿠!` 알림을 실제 미로 영역 중심(`mazeZone`) 기준으로 정렬
- Render 자동배포 필터를 실행 코드와 빌드 영향 파일 기준으로 정리

### Fixed

- 레이스 로그 패널의 텍스트 색상 대비 수정
- 가짜 골 알림의 크기와 위치 보정을 미로 판 기준으로 안정화

## [0.1.0] - 2026-03-24

첫 공개 릴리즈입니다.

### Added

- 실시간 멀티 레이스 기본 게임 루프 추가
- 방 생성, 방 목록, 입장, 대기 상태, 경기 시작 흐름 추가
- `Socket.IO` 기반 authoritative 서버 동기화 추가
- 경기 중 연결 끊김 복구와 유예 시간 처리 추가
- 방장 위임, 강제 종료, 시야 크기 조절 등 방 운영 기능 추가
- 관리자 맵 API와 웹 기반 맵 편집 화면 추가
- 커스텀 맵 저장소와 `data/maps.json` 영속 저장 추가
- 탐험한 타일 기억 안개와 플레이어 마커 식별 강화 추가
- Dockerfile, Docker Compose, Docker 제어 스크립트 추가
- LAN 환경에서 개발 서버 접속 가능 설정 추가
- 루트 README와 릴리즈 문서 추가

### Changed

- 미로 구성과 기본 플레이 맵 난이도 조정
- 플레이어 패널, 색상 팔레트, 게임 화면 밀도와 레이아웃 개선
- 대기 상태 프리뷰와 시야 렌더링 규칙 정리
- 프로젝트 문서와 명세를 현재 구현 기준으로 동기화

### Fixed

- 시작 전 이동과 입력 처리 안정화
- 대기/플레이 레이아웃 흔들림과 스크롤 문제 수정
- 서버 기준 플레이어 모양 배정으로 식별 불일치 수정
- 기억 타일 명도 조정으로 시야 구분 강화
- 테스트 환경과 E2E 시나리오 보정
