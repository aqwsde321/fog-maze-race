# Fog Maze Race v0.1.0 릴리즈 노트

릴리즈 날짜: `2026-03-24`
상태: 초안
예정 태그: `v0.1.0`

## 요약

`v0.1.0`은 Fog Maze Race의 첫 공개 릴리즈입니다. 실시간 멀티 미로 레이스의 핵심 MVP 흐름과 관리자 맵 편집, Docker 실행, LAN 개발 접속까지 포함합니다.

## 포함된 내용

### 게임 플레이

- 닉네임 입력 후 방 생성/입장
- 카운트다운 후 미로 레이스 시작
- 서버 authoritative 이동 판정과 완주 순위 기록
- 결과 화면 표시 후 대기 상태 복귀
- 기본 시야 `5x5`와 방장 전용 시야 크기 변경
- 실시간 방 채팅

### 실시간 복구와 방 운영

- 연결 끊김 유예 시간 기반 재접속 복구
- 유예 초과 시 자동 이탈 처리
- 방장 위임
- 방장 전용 경기 시작, 시야 크기 변경, 강제 종료
- `normal`, `bot_race` 방 모드
- `bot_race`에서 사람 관전자 + 봇 레이서 운영

### 봇과 테스트 지원

- 방장 UI에서 이름 기반 봇 추가/제거
- 빠른 최단 경로 봇과 탐험형 봇 제공
- 탐험형 전략 `frontier`, `tremaux` 지원
- `pnpm race:join`, `pnpm race:explore`, `pnpm race:fill` CLI 도구 제공

### 맵과 관리자 기능

- 기본 플레이 맵 세트 제공
- 관리자 맵 조회/생성/수정/삭제
- 맵 저장소를 통한 커스텀 맵 영속화

### 개발/운영 지원

- `pnpm dev` 기반 로컬 개발
- 같은 공유기에서 접속 가능한 LAN 개발 서버
- Dockerfile, Compose, 제어 스크립트 제공
- 루트 README 및 온보딩 문서 추가

## 실행 방법

### 개발 모드

```bash
pnpm install
pnpm dev
```

- 웹: `http://127.0.0.1:4173`
- 서버 헬스 체크: `http://127.0.0.1:3000/health`

### Docker / Compose

```bash
pnpm docker:up
pnpm docker:ps
pnpm docker:logs
```

- 기본 주소: `http://127.0.0.1:3300`

## 검증 기준

릴리즈 전 최소 확인 항목:

- `pnpm typecheck`
- `pnpm test`
- 필요 시 `pnpm test:e2e`
- `pnpm docker:up` 후 헬스 체크 응답 확인

## 현재 제약 사항

- 단일 인스턴스 전제
- 런타임 방/세션/경기 상태는 서버 메모리 기반
- 관리자 맵은 `data/maps.json` 파일 기반 저장
- 다중 인스턴스 확장과 외부 세션 저장소는 아직 미포함
- 탐험형 전략은 현재 `frontier`, `tremaux` 두 종류만 제공
